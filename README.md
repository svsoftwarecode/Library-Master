## Description
**Library Master** is a **C# desktop application** designed to manage library operations efficiently.  
It allows librarians to manage library records, track members, and handle borrowing/return processes through a user-friendly interface with **SQL database integration**.

---

## Features
1. Add and edit books  
2. View all books, issued books, and their counts  
3. View available books count  
4. Add, view, edit, and delete members  
5. See member's book details and payment details  
6. Issue books  
7. Return books  
8. Payments (new membership, renew membership, fines)  
9. View payment details  
10. Add, view, edit, and delete staff members
